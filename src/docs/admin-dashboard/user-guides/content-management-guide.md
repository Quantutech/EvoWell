# Content Management Guide

## Overview
The Content Management system allows you to create, review, and publish rich content including Blog Posts, Testimonials, and Resources.

## Key Features

### 1. Rich Text Editor
A modern, block-based editor (Tiptap) that supports:
- **Formatting:** Bold, Italic, Headings (H1, H2), Bullet Lists.
- **Media:** Embed images directly from the Media Library.
- **Live Metrics:** Real-time word count and reading time estimation.

### 2. Workflow System
Content moves through a defined lifecycle to ensure quality:
1.  **Draft:** Initial creation state.
2.  **Internal Review:** Submitted for peer review.
3.  **SEO Review:** Optimization check.
4.  **Legal Review:** Compliance check (if required).
5.  **Approved:** Ready for publication.
6.  **Published:** Live on the platform.

### 3. SEO Tools
Optimize your content for search engines *before* you publish:
- **Health Score:** 0-100 rating based on best practices.
- **Google Preview:** See exactly how your post will look in search results.
- **Actionable Tips:** Real-time suggestions (e.g., "Add an image with alt text").

### 4. Media Library
A central repository for all platform assets:
- **Grid View:** Visual browsing of images and documents.
- **Batch Upload:** Drag-and-drop multiple files at once.
- **Tagging:** Organize assets by clinical topic or campaign.
