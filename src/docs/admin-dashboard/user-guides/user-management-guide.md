# User Management Guide

## Overview
The User Management module provides a unified interface for managing all platform users, including Administrators, Practitioners, and Clients.

## Key Features

### 1. Unified User Table
- **Global Search:** Quickly find users by name, email, or role.
- **Advanced Filters:** Filter by role (Provider, Client), status (Active, Pending), and registration date.
- **Sorting:** Click any column header to sort the data.

### 2. Bulk Actions
Perform actions on multiple users simultaneously:
1. Select users using the checkboxes.
2. Choose an action from the "Bulk Actions" dropdown (e.g., Delete, Export).
3. Confirm the operation.
> **Note:** Destructive actions like deletion support "Undo" for a few seconds after execution.

### 3. Inline Quick Edit
Update simple fields without opening the full profile:
- Click on a user's **Role** or **Status** in the table.
- Select the new value from the dropdown.
- Changes are saved automatically.

## Role-Specific Directories

### Practitioners Tab
Dedicated view for managing healthcare providers.
- **License Verification:** View and verify medical license numbers.
- **Specialties:** Manage clinical tags.
- **Vetting Status:** Approve or reject provider applications.

### Clients Tab
View for managing patient accounts.
- **Intake Status:** Track onboarding progress.
- **Document Access:** View uploaded health records.
