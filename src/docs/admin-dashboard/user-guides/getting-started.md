# Getting Started with EvoWell Admin

## Introduction
Welcome to the EvoWell Admin Dashboard. This guide will help you navigate the interface and understand key concepts.

## Access Levels
Your view of the dashboard depends on your assigned role:
- **Super Admin:** Full access to all settings and data.
- **Content Manager:** Access to Blog, Media, and SEO tools.
- **User Manager:** Access to Practitioner and Client directories.
- **Support Agent:** Access to Tickets and Message Center.

## Dashboard Overview
The main dashboard provides a real-time snapshot of platform health:
1.  **Metric Cards:** Total Users, Active Providers, Open Tickets.
2.  **Growth Chart:** Visualization of user acquisition over time.
3.  **Recent Activity:** Live feed of important system events.

## Navigation
Use the sidebar to access different modules:
- **User Management:** Manage Practitioners, Clients, and Applications.
- **Content:** Create blogs, manage media, and review testimonials.
- **Support:** Handle tickets and user inquiries.
- **System:** Configure platform settings and view audit logs.

> **Tip:** Press `Cmd+K` (or `Ctrl+K`) to open the **Command Palette** for quick navigation and actions.
